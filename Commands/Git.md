### switch branch (new)
```git
git switch
```

### add files 
```git
git add <files>
```

### commit
```git
git commit -m "message_here"
```

### push, pull
```git 
git push <branch> 
git pull <branch>
```

### git ignore (directories/file)
inside .gitignore file, put directory / file names

### check logs
```git
git log
```

### show branches
```git
git branch
``` 

### change to other versions of your application
```git
git checkout <hash>/<branch>
```
