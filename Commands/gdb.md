> Run GDB
```bash
gdb <program_name>
```

> set breakpoint at main
```bash
break *main
```

> run program
```bash
run
```

> dissamble the function into assembly
```bash
dissamble main
```

> set assembly flavor to intel
```bash
set disassembly-flavor intel
```

> check map memory

```bash
info proc mappings
```
